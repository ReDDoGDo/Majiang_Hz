# 杭州麻将需求描述

## 一、项目概述

### 1.1 项目背景
杭州麻将是一款流行于浙江杭州地区的麻将游戏，具有独特的规则和玩法。本项目旨在开发一款可在Steam平台发布的单机/联机麻将游戏，供玩家休闲娱乐和练习使用。

### 1.2 目标用户
- 杭州麻将爱好者
- 麻将规则学习者
- 休闲游戏玩家
- Steam平台用户

### 1.3 项目目标
- 实现完整的杭州麻将规则
- 提供流畅的游戏体验
- 支持单机AI对战
- 支持多人联机对战（可选）
- 发布到Steam平台

## 二、杭州麻将规则

### 2.1 基础设置
- **牌数**: 136张（万、条、筒各36张 + 东南西北风16张 + 中发白8张）
- **人数**: 4人
- **庄家**: 第一局随机确定，之后胡牌者坐庄
- **手牌**: 每人13张，庄家14张
- **骰子**: 2颗

### 2.2 游戏流程

#### 2.2.1 洗牌与发牌
1. 洗牌：随机打乱136张牌
2. 掷骰子：确定开牌位置
3. 发牌：庄家14张，其他玩家13张
4. 翻宝牌：翻开一张牌作为宝牌（财神）

#### 2.2.2 游戏进行
1. 庄家先出牌
2. 其他玩家按逆时针顺序进行
3. 每回合：摸牌 → 杠/碰/吃/胡 → 出牌
4. 直到有人胡牌或牌墙摸完（流局）

### 2.3 牌型组合

#### 2.3.1 基本牌型
- **顺子**: 同花色连续三张（如：123万）
- **刻子**: 相同三张（如：111万）
- **杠子**: 相同四张（如：1111万）
- **对子**: 相同两张（如：11万）

#### 2.3.2 胡牌条件
- 4个顺子/刻子/杠子 + 1个对子
- 7个对子（七对子）
- 特殊牌型（详见特殊胡牌）

### 2.4 操作规则

#### 2.4.1 吃牌
- 只能吃上家打出的牌
- 组成顺子
- 杭州麻将通常**不允许吃牌**

#### 2.4.2 碰牌
- 可碰任何玩家打出的牌
- 组成刻子
- 碰牌后必须打出一张牌

#### 2.4.3 杠牌
- **明杠**: 碰牌后摸到第四张，或直接杠别人打出的牌
- **暗杠**: 自己手中有四张相同的牌
- 杠牌后需摸一张牌（补张）
- 杠牌后翻新宝牌

#### 2.4.4 胡牌
- **自摸**: 自己摸到的牌胡牌
- **点炮**: 别人打出的牌胡牌
- **抢杠胡**: 杠牌时被胡牌

### 2.5 宝牌（财神）规则

#### 2.5.1 宝牌确定
- 翻开一张牌，其下一张为宝牌
- 万、条、筒按数字顺序循环（9的下一张是1）
- 风牌：东南西北循环
- 箭牌：中发白循环

#### 2.5.2 宝牌作用
- 宝牌可以替代任何牌
- 宝牌本身不能被替代
- 计算番数时，宝牌有额外加成

### 2.6 番数计算

#### 2.6.1 基础番型
- **平胡**: 1番 - 基础胡牌牌型
- **自摸**: 1番
- **宝牌**: 每张宝牌1番
- **杠开**: 1番 - 杠牌后立即胡牌
- **抢杠**: 1番 - 抢杠胡

#### 2.6.2 特殊番型
- **七对子**: 2番 - 7个对子
- **清一色**: 6番 - 只有万/条/筒一种花色
- **混一色**: 3番 - 一种花色 + 风牌/箭牌
- **对对胡**: 4番 - 4个刻子/杠子
- **清对对**: 8番 - 清一色 + 对对胡
- **混对对**: 5番 - 混一色 + 对对胡
- **全求人**: 2番 - 单吊一张牌胡牌
- **大三元**: 8番 - 中发白三组刻子/杠子
- **小三元**: 6番 - 中发白两组刻子 + 一对
- **字一色**: 10番 - 全是风牌/箭牌
- **清幺九**: 8番 - 只有1和9
- **十三幺**: 10番 - 特殊牌型（需确认杭州麻将是否支持）

#### 2.6.3 番数叠加
- 多个番型可叠加计算
- 最大封顶：根据房间设置（建议10番封顶）

### 2.7 计分规则

#### 2.7.1 基础计分
- 底分：1分
- 番数对应倍数：
  - 1番: 2倍
  - 2番: 4倍
  - 3番: 8倍
  - 4番: 16倍
  - 5番及以上: 封顶倍数

#### 2.7.2 输赢计算
- **自摸**: 其他三家各付给胡牌者
- **点炮**: 点炮者单独付给胡牌者
- **杠牌**: 杠牌者从其他三家收取分数

### 2.8 特殊规则

#### 2.8.1 杭州麻将特色
- **不能吃牌**: 只能碰、杠、胡
- **宝牌翻倍**: 持有宝牌数量影响番数
- **连庄规则**: 胡牌者坐庄，流局则庄家继续
- **一炮多响**: 一张牌可被多人胡牌

#### 2.8.2 流局规则
- 牌墙摸完无人胡牌
- 流局后庄家继续坐庄
- 可选择是否查花猪（未确认规则）

## 五、用户故事

### 5.1 单机游戏
**作为一个玩家，我想要和AI对战，以便随时练习麻将技巧。**

验收标准：
- 可以选择AI难度
- AI行为合理，有挑战性
- 游戏流程顺畅

### 5.2 规则学习
**作为一个新手，我想要查看规则说明，以便了解杭州麻将的玩法。**

验收标准：
- 规则说明完整清晰
- 有牌型示例
- 易于理解

### 5.3 联机对战
**作为一个玩家，我想要和朋友联机对战，以便一起娱乐。**

验收标准：
- 可以创建/加入房间
- 网络连接稳定
- 游戏体验流畅

### 5.4 成就收集
**作为一个玩家，我想要解锁Steam成就，以便获得成就感。**

验收标准：
- 成就设计合理
- 解锁提示清晰
- 与Steam同步
