## UI设计规范

### 颜色方案
- 主色调：绿色（麻将桌风格）
- 辅助色：金色、红色
- 背景色：深绿色、棕色
- 文字色：白色、黑色

### 字体规范
- 标题字体：微软雅黑 Bold
- 正文字体：微软雅黑 Regular
- 数字字体：Arial Bold

### 按钮规范
- 正常状态：绿色背景
- 悬停状态：亮绿色背景
- 点击状态：深绿色背景
- 禁用状态：灰色背景

### 布局规范
- 适配分辨率：1920x1080（主要）、1280x720（最低）
- UI缩放：Canvas Scaler
- 锚点：合理设置

## 技术实现

### UI框架
- 使用Unity UGUI系统
- Canvas设置：Screen Space - Overlay
- 分辨率适配：Scale With Screen Size

### UI动画
- 使用DOTween实现动画
- 按钮点击动画
- 界面切换动画
- 牌的移动动画

### UI交互
- 使用EventSystem处理点击事件
- 使用Button组件
- 使用Toggle、Slider等UI组件

### UI管理
- 使用UIManager统一管理
- 界面栈管理
- 界面缓存池

## 数据结构

### UIState（UI状态）
```csharp
public enum UIState
{
    MainMenu,
    Game,
    Settings,
    Result,
    Rules
}
```

### UIManager（UI管理器）
```csharp
public class UIManager : Singleton<UIManager>
{
    private Stack<UIState> _uiStack;
    private Dictionary<UIState, GameObject> _uiCache;
    
    public void PushUI(UIState state);
    public void PopUI();
    public void ChangeUI(UIState state);
}
```
