## 核心算法

### 胡牌判断算法（核心难点）

#### 基础思路
1. 检查是否满足基本胡牌条件（4个顺子/刻子 + 1个对子）
2. 检查是否为七对子
3. 检查特殊牌型
4. 考虑宝牌替代

#### 算法实现
```csharp
public bool CheckHu(List<Tile> handTiles, Tile wildCard)
{
    // 移除宝牌，记录数量
    int wildCardCount = handTiles.Count(t => t.IsWildCard);
    var normalTiles = handTiles.Where(t => !t.IsWildCard).ToList();
    
    // 检查七对子
    if (CheckSevenPairs(normalTiles, wildCardCount))
        return true;
    
    // 检查标准胡牌
    if (CheckStandardHu(normalTiles, wildCardCount))
        return true;
    
    return false;
}
```

### 番数计算算法

#### 番型识别
1. 清一色：只有一种花色
2. 混一色：一种花色 + 字牌
3. 对对胡：全是刻子/杠子
4. 七对子：7个对子
5. 其他特殊牌型

#### 番数叠加
- 多个番型可叠加
- 最大封顶（如10番）
